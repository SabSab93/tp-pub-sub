# Front Vue 3 (Vite) — Socket.IO + REST

## Installation
```bash
npm i
```

## Variables d'environnement
Créer `.env` (déjà fourni) :
```
VITE_API_URL=http://localhost:3000
VITE_WS_URL=http://localhost:3000
```

## Lancer en dev
```bash
npm run dev
```

## Build
```bash
npm run build
npm run preview
```
